# ВИСНОВКИ

Під час проходження виробничої практики було успішно виконано програму практики та реалізовано пункти індивідуального завдання, пов’язані з проектуванням та розробкою веб-застосунку для системи обліку робочого часу працівників.

У ході роботи були досягнуті наступні результати відповідно до поставлених завдань:

1. **Проектування серверної частини:** Розроблено архітектуру бекенду з використанням платформи Node.js та фреймворку Express.js, що забезпечує обробку HTTP-запитів та маршрутизацію. Реалізовано модульну структуру з чітким розділенням відповідальності між різними компонентами системи (аутентифікація, управління запитами, адміністрування, новини, свята).

2. **Робота з даними:** Реалізовано бізнес-логіку системи та налаштовано взаємодію з нереляційною базою даних MongoDB Atlas через ORM Prisma. Створено моделі даних для збереження інформації про користувачів, запити на відпустку, корпоративні новини та календар свят. Prisma забезпечила типобезпеку та автоматичну генерацію типів TypeScript.

3. **Реалізація REST API:** Спроектовано та реалізовано RESTful API з використанням Express.js та TypeScript. Розроблено endpoints для аутентифікації, управління запитами на відпустку, адміністративних функцій, публікації новин та управління святами. Реалізовано систему аутентифікації на базі JWT-токенів та розмежування прав доступу (ролі працівник/адміністратор).

4. **Розробка клієнтської частини:** Створено інтуїтивний інтерфейс користувача на базі Next.js 14 та React з використанням TypeScript. Реалізовано компоненти для реєстрації та входу, подачі запитів на відпустку, перегляду статистики вихідних днів, інтегрованого календаря з відображенням свят та періодів відпусток. Забезпечено підтримку багатомовності (українська/англійська) та адаптивний дизайн на базі Tailwind CSS.

5. **Бізнес-логіка та автоматизація:** Впроваджено алгоритм автоматичного розрахунку нарахованих вихідних днів на основі дати початку роботи працівника (1.5 дня відпустки на місяць роботи, 10 днів лікарняних на рік). Реалізовано систему нарахування додаткових вихідних днів адміністратором з веденням історії нарахувань.

6. **Забезпечення якості та безпеки:** Забезпечено типобезпеку через TypeScript та Prisma ORM, реалізовано двохрівневу валідацію вхідних даних на клієнті та сервері, впроваджено хешування паролів через bcryptjs та захист від неавторизованого доступу через JWT middleware. Реалізовано централізовану обробку помилок з логуванням та перекладом повідомлень.

7. **Адміністративна панель:** Розроблено повнофункціональну адміністративну панель для управління користувачами, схвалення/відхилення запитів на відпустку та лікарняний, нарахування додаткових вихідних днів, публікації корпоративних новин та налаштування календаря свят.

Для вдосконалення роботи досліджуваного об'єкта пропонується в майбутньому впровадити наступні покращення:

- **Кешування даних:** Впровадження кешування на рівні Redis для зменшення навантаження на базу даних при частому зверненні до календаря свят та статистики користувачів.

- **Автоматизоване тестування:** Розробка модульних та інтеграційних тестів з використанням Jest для backend та Vitest для frontend для забезпечення надійності системи при внесенні змін.

- **Контейнеризація:** Налаштування Docker та Docker Compose для забезпечення консистентності середовища розробки та спрощення розгортання на різних платформах.

- **Розширення функціоналу:** Додавання системи сповіщень (email/повідомлення) про зміну статусу запитів, експорт звітів у форматі PDF, інтеграція з календарями (Google Calendar, Outlook) для синхронізації відпусток.

- **Оптимізація продуктивності:** Впровадження пагінації для великих списків запитів та новин, оптимізація запитів до бази даних через індекси Prisma, lazy loading для компонентів календаря.

Результати роботи підтверджують виконання індивідуального плану практики та засвоєння навичок розробки сучасних веб-застосунків на базі стеку MERN (MongoDB, Express.js, React, Node.js) з використанням TypeScript, Prisma ORM та Next.js. Створена система забезпечує автоматизацію обліку робочого часу, ефективне управління відпустками та лікарняними, а також забезпечує зручний інтерфейс для взаємодії між працівниками та адміністрацією.
